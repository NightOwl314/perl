NAME

Test::Harness::Assert - simple assert

SYNOPSIS

  ### FOR INTERNAL USE ONLY ###
  use Test::Harness::Assert;
  assert( EXPR, $name );

DESCRIPTION

A simple assert routine since we don't have Carp::Assert handy.

For internal use by Test::Harness ONLY!

Functions
