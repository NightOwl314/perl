NAME

IPC::SysV - SysV IPC constants

SUPPORTED PLATFORMS

  • Linux
  • Solaris

SYNOPSIS

    use IPC::SysV qw(IPC_STAT IPC_PRIVATE);

DESCRIPTION

IPC::SysV defines and conditionally exports all the constants defined in your system include files which are needed by the SysV IPC calls.

ftok( PATH, ID )
Return a key based on PATH and ID, which can be used as a key for msgget, semget and shmget. See ftok

SEE ALSO

the IPC::Msg manpage, the IPC::Semaphore manpage, ftok
