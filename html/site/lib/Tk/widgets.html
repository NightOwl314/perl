NAME

Tk::widgets - preload widget classes

SYNOPSIS

  use Tk::widgets qw(Button Label Frame);

DESCRIPTION

Does a 'require Tk::Foo' for each 'Foo' in the list. May speed startup by avoiding AUTOLOADs.
